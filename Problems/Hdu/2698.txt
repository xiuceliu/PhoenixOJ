<h1 style="color:#1A5CC8">URL</h1>
<font><b><span style="font-family:Arial;font-size:12px;font-weight:bold;color:green">Time Limit: 2000/1000 MS (Java/Others)&#160;&#160;&#160;&#160;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 449&#160;&#160;&#160;&#160;Accepted Submission(s): 117<br></span></b></font><br><br><div class="panel_title" align="left">Problem Description</div> <div class="panel_content">WHU ACM Team is working on a brand new web browser named "Whu-Super-Browser". You're in response for a  <br>powerful feature: recording the previous addresses. Moreover, when a string is inputted, the browser will display all the  <br>addresses start with it. The addresses shall be sorted by visited times, in descending order. This feature is very useful to  <br>users. <br>Can you complete it? <br><br>There're two kinds of operations: <br>Visit [url_str] : visit a website with the URL: [url_str]. <br>Display [str] : display all addresses start with [str] and sort them by visited times, in descending order. If two addresses  <br>have the same visited times, then sort them in the lexicographical order. </div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Input</div> <div class="panel_content">The input consists of multiple test cases. The first line of input contains an integer T, which is the number of test cases. <br><br>Each test case is on several lines. <br>The first line of each test case consists of an integer N. <br>Each of the following N lines consists of an operation, Visit or Display. <br><br>[Technical Specification] <br>T is an integer, and T &lt;= 10. <br>N is an integer, and 1 &lt;= N &lt;=100. <br>There's NO blank line between test cases. <br>[url_str] and [str] only contains lower case letters 'a' - 'z', '.', '/', ':'. <br>The length of [url_str] and [str] is greater than 0 and won't exceed 100. </div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Output</div> <div class="panel_content">For each test case, display addresses as required, each address on a separated line.  <br>Add a blank line after each 'Display' operation. </div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Sample Input</div>
<div class="panel_content"><pre><div style="font-family:Courier New,Courier,monospace;">1 
10 
Visit http://acm.whu.edu.cn 
Visit http://acm.pku.edu.cn 
Visit http://acm.timus.ru 
Visit http://acm.whu.edu.cn 
Visit http://acm.whu.edu.cn 
Visit http://acm.pku.edu.cn 
Display http://acm 
Visit baidu.com 
Visit www.whu.edu.cn 
Display b </div></pre></div>
<div class="panel_bottom">&#160;</div>
