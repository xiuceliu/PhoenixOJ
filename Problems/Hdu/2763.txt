<h1 style="color:#1A5CC8">The Two Note Rag</h1>
<font><b><span style="font-family:Arial;font-size:12px;font-weight:bold;color:green">Time Limit: 2000/1000 MS (Java/Others)&#160;&#160;&#160;&#160;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 51&#160;&#160;&#160;&#160;Accepted Submission(s): 27<br></span></b></font><br><br><div class="panel_title" align="left">Problem Description</div> <div class="panel_content">Since most computers are binary machines, both powers of two and problems that involve only two values are important to computer scientists. The following problem has to do with powers of two and the digits 1 and 2.<br><br>Some powers of two as decimal values, such as 2<sup>9</sup> = 512 and 2<sup>89</sup> = 618,970,019,642,690,137,449,562,112 end in a string of digits consisting only of 1's and 2's (12 for 2<sup>9</sup> and 2112 for 2<sup>89</sup> ). In fact, it can be proved that:<br><br>For every integer R, there exists a power of 2 such that 2<sup>K</sup> uses only the digits 1 and 2 in its last R digits.<br><br>This is shown a bit more clearly in the following table: <br><br><center><img src="../../../data/images/C178-1008-1.jpg"></center>
<br><br>Your job is to write a program that will determine, for given R, the smallest K such that 2<sup>K</sup>  ends in a string of R digits containing only 1's and 2's. </div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Input</div> <div class="panel_content">The first line of the input contains a single decimal integer, N, 1 ≤ N ≤ 50, the number of problem data sets to follow. Each data set consists of a single integer R, 1 ≤ R ≤ 20, for which we want a power of 2 ending in a string of R 1's and 2's. </div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Output</div> <div class="panel_content">For each data set, you should generate one line of output with the following values: The data set number as a decimal integer (start counting at one), a space, the input value R, another space, and the smallest value K for which 2<sup>K</sup> ends in a string of R 1's and 2's. </div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Sample Input</div>
<div class="panel_content"><pre><div style="font-family:Courier New,Courier,monospace;">6 
1 
2 
4 
5 
7 
15 </div></pre></div>
<div class="panel_bottom">&#160;</div>
<br><div class="panel_title" align="left">Sample Output</div>
<div class="panel_content"><pre><div style="font-family:Courier New,Courier,monospace;">1 1 1 
2 2 9 
3 4 89 
4 5 589 
5 7 3089 
6 15 11687815589  </div></pre></div>
<div class="panel_bottom">&#160;</div>
